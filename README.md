# Portafolio - Angel Lugo

Portafolio personal como Desarrollador Full Stack, construido con Next.js 16, Tailwind CSS y shadcn/ui.

## Características

- **Next.js 16** con App Router
- **shadcn/ui** para componentes
- **Tema Charcoal + Esmeralda** (claro y oscuro)
- **Responsive** y optimizado para móvil
- Secciones: Inicio, Sobre mí, Experiencia, Proyectos, Habilidades, Formación, Contacto

## Desarrollo

```bash
npm install
npm run dev
```

Abre [http://localhost:3000](http://localhost:3000) en el navegador.

## Build

```bash
npm run build
npm start
```
